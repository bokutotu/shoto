## Environment

- Work inside `nix develop`.
- `lefthook` runs `./format.sh` and `cabal test all` on pre-commit for touched Haskell files.

## Commands

- Test: `cabal test all`
- CUDA Test: `cabal --project-file=cabal.project.cuda test all`
- Lint: `hlint .`
- Apply HLint refactors: `./hlint-refactor.sh`
- Format Haskell: `./format.sh`
- cabal files are generated by `hpack`
- use `nix develop .#cuda` for CUDA-enabled builds and tests with the system CUDA installation

## Haskell Style

- Prefer record-dot syntax (`x.field`) in new or edited code.

## Documentation
